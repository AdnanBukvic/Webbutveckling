Adnan Bukvic

Jag är en 16 år gammal kille med föräldrar som flyttade till Sverige från forna Jugoslavien 1995.

Jag gillar att hålla på runt och i datorer och mobiltelefoner. Jag gillar även att lyssna på musik -- mestadels Ariana Grande och liknande artister.

EEDAT

Med tanke på att jag gillar datorere, el och mobiltelefoner var det här nästanen självklar linje för mig. Jag ville också lära mig kodning och HTML mer detaljerat. HTML och CSS kan jag redan en hel del om, men bara grunderna. CSS är jag rätt så van med. Ibland på min fritid brukar jag ta CSS dokument, redigera dem och publicera de som forks. Det sensate projektet jag har är ett CSS tema för Firefox på Linux så det ser ut som Safari från en Mac. Jag gjorde en fork av den så att den fungerear med den nyaste Firefox.
